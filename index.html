O técnico em informática é um profissional que tem entre suas responsabilidades a montagem, acabamento e manutenção de computadores, além da identificação e correção de problemas em aparelhos eletrônicos.
O que é o curso de TI? O curso de TI, na verdade, é um termo comumente usado para fazer referência às formações na área de Tecnologia da Informação. Para entender quais são elas e o conteúdo estudado, é preciso, primeiro, compreender esse campo de conhecimento.
No mercado de trabalho, o profissional de TI encontra boas oportunidades no Brasil e no exterior, já que se trata de uma área aquecida e com carência de qualificação.
Em linhas gerais, ele oferece qualificação para quem deseja trabalhar com a criação de soluções em sistemas de informação de dados. Seu enfoque pode variar de acordo com a instituição de ensino e englobar desenvolvimento de softwares ou até mesmo proteção e gestão de dados.



O curso de TI é cheio de disciplinas de exatas, mas também tem muitas disciplinas de gestão e negócios. Veja a lista das disciplinas mais comuns do curso de TI, elas podem ser vistas em instituições como a Cruzeiro do Sul Virtual (a distância): Administração de Banco de Dados. Comércio Eletrônico. Contabilidade.
No geral, a definição de TI não inclui tecnologias incorporadas, que não geram dados para o uso das empresas.
Então TI são todos os tipos de soluções tecnológicas, que usamos diariamente dentro de uma empresa, ou seja, tudo começa a partir do seu acesso a um computador em rede para outros lugares (dentro de uma empresa).



De longe, o sistema de controle de versão moderno mais usado no mundo hoje é o Git. O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux. Um número impressionante de projetos de software depende do Git para controle de versão, incluindo projetos comerciais e de código-fonte aberto.

Em suma, o GitHub é um serviço baseado em nuvem que hospeda um sistema de controle de versão (VCS) chamado Git. Ele permite que os desenvolvedores colaborem e façam mudanças em projetos compartilhados enquanto mantêm um registro detalhado do seu progresso.
